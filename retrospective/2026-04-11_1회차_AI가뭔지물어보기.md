# 1단계 1회차 회고 — AI가 뭔지 물어보기

> **날짜:** 2026-04-11 (금)
> **소요 시간:** 약 60분
> **강사:** Kay / **학생:** 자녀 (11세, 초5)

---

## 수업 진행 요약

### 1. 도입 — AI의 개념과 한계 (대화 시작)
- AI는 **100% 정답을 주지 않는다**는 점을 인지시킴
- AI의 학습 과정이 사람과 다름을 설명
  - 사람: 책, 매체를 통해 앉아서 공부
  - AI: 방대한 데이터에서 패턴을 학습하여 응답을 생성
- **보완 필요**: "여러 자료를 종합하여 voting" 비유를 사용했으나, 보다 정확한 설명은 "패턴을 학습하고 다음 단어를 예측하는 방식". 다음 수업에서 자연스럽게 보정 필요

### 2. 활동 1 — 수학 문제 속도 비교
- 단순 계산에서 AI의 월등한 성능을 체험
- 계획대로 진행됨, 효과적

### 3. 활동 2 — 취향 질문 (색깔 선호도)
- AI에게 색깔 선호도를 질문 → AI에게 취향이 없음을 확인
- **추가 활동 (계획 외)**: 통계를 조회하여 "사람들이 어떤 색을 더 좋아하는지" 프롬프팅하는 방법을 소개
  - AI가 취향은 없지만 통계 기반 답변은 가능하다는 차이를 학습
  - 프롬프팅 스킬에 대한 첫 접점이 됨

### 4. 아이 자발적 탐구 — 존댓말 vs 반말
- Kay가 "한국어 모델에서 존댓말 사용 시 더 좋은 결과가 나올 수 있다"는 경험담 공유 (SOTA 모델링 관련)
- 아이가 Claude에게 "반말이 좋아? 존댓말이 좋아?" 질문
- AI의 언어 처리 방식에 대한 호기심으로 연결됨

### 5. 아이 자발적 탐구 — 의미 없는 입력 테스트
- `\/\/\/\/\/\` 같은 의미 없는 프롬프트를 입력
- AI가 "무슨 의미인지 모르겠다"고 응답
- → **명확한 질문이 중요하다**는 점을 강조하는 계기로 활용

---

## 아이 노트 (원문)

> "AI가 존댓말을 쓰면 좋아할 것 같은데 반말을 쓰는게 더 편하다고 해서 그게 제일 신기했다."

---

## 잘된 점

1. **자발적 탐구 발생**: 존댓말/반말 질문, 의미 없는 프롬프트 테스트 등 계획에 없던 실험을 아이가 스스로 시도
2. **프롬프팅 스킬 자연 습득**: 색깔 선호도 → 통계 조회 프롬프팅으로 자연스럽게 확장
3. **핵심 메시지 전달 성공**: "AI는 100% 정답이 아니다", "명확하게 질문해야 한다"

## 개선할 점

1. **AI 학습 방식 설명 보정 필요**
   - "voting" 비유 → "패턴 학습 + 다음 단어 예측" 비유로 보정
   - 다음 수업 도입부에서 자연스럽게 "지난번에 AI가 어떻게 공부한다고 했지?" 로 복습하며 보정 가능

2. **자발적 탐구를 교육자료에 반영**
   - 아이가 보인 "의미 없는 입력 테스트"는 매우 좋은 학습 활동
   - 강사용/학생용 HTML에 해당 활동을 추가하면 다른 아이들(또는 재수업 시)에도 활용 가능

3. **프롬프팅 기법 소개 시점 조정**
   - 통계 기반 프롬프팅은 3회차(프롬프트 실험)와 연결되는 내용
   - 1회차에서 가볍게 맛보기 → 3회차에서 본격 심화하는 흐름으로 교육자료 정리 가능

## 교육자료 개선 제안

### 강사용 자료 (1회차 슬라이드)
1. **AI 학습 방식 설명 가이드 추가**: "패턴 학습" 비유 예시를 강사 팁으로 포함
2. **보너스 활동 추가**: "의미 없는 프롬프트 실험" — `asdfghjkl`, `\/\/\/\` 등 입력해보기 → 명확한 질문의 중요성 체험

### 학생용 자료 (1회차 페이지)
1. **질문 목록에 추가 질문 제안**:
   - "AI야, 반말이 좋아? 존댓말이 좋아?" (AI의 언어 처리 탐구)
   - 이상한 문자 입력해보기 활동 (명확한 질문 중요성)
2. **프롬프팅 팁 카드 추가**: "같은 질문이라도 어떻게 물어보느냐에 따라 답이 달라져!" 힌트
