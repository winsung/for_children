🧺
단계 1
바구니 움직이기
키보드로 바구니를 조종해보자!
⏱ 15분
🛠️ 오늘 쓸 도구 — 엔트리
🟢
엔트리 접속하기
playentry.org → 만들기 → 작품 만들기
↗
엔트리봇은 지우고, "바구니"와 "별" 오브젝트를 추가해서 시작하자.
🎯 미션
바구니가 ← → 키로 움직이게 만들자.
📋 과제 — 이걸 해내면 성공!
①
바구니를 화면 아래쪽 가운데에 두기
게임이 시작될 때 항상 같은 자리에서 출발하게
②
왼쪽 키를 누르면 왼쪽으로
← 화살표 키로 바구니가 왼쪽으로
③
오른쪽 키를 누르면 오른쪽으로
→ 화살표 키로 바구니가 오른쪽으로
💡 힌트 1 — 카테고리만
"키를 눌렀을 때" 같은 블록은 어느 카테고리에 있을까?
(→ 시작 카테고리를 열어봐!)
💡 힌트 2 — 직접 실험
왼쪽으로 가려면 x좌표를 +로 바꿀까, -로 바꿀까?
둘 다 해보고 어느 쪽이 왼쪽인지 직접 확인해봐!
🤫 약속
정답 블록은 안 알려줄게 — 카테고리만 힌트야. 블록은 네가 찾는 거야!
✅ 과제 체크리스트
✓
바구니가 시작할 때 아래쪽 가운데에 있다
✓
← 키로 바구니가 왼쪽으로 움직인다
✓
→ 키로 바구니가 오른쪽으로 움직인다
✓
▶ 실행해서 좌우로 잘 움직이는지 확인했다
✏️ 이 블록이 뭐 하는 건지 말로 설명해보기
"키를 눌렀을 때" 블록이 무슨 일을 하는지, 아빠한테 한 문장으로 설명해봐.
🆘 막혔을 때 — 3단계 규칙
① 10분 혼자 → 어느 카테고리에 있을지 직접 둘러보기
② AI에 힌트만 → "정답 말고 어느 카테고리인지만!"
③ 직접 끼우고 설명 → 말로 설명 못 하면 다시!
🔢
단계 2
점수 만들기
별을 받으면 숫자가 올라가게!
⏱ 10분
🎯 미션
별을 받으면 숫자가 올라가게 하자.
📋 과제 — 이걸 해내면 성공!
①
점수를 보여줄 무언가 만들기
화면에 숫자가 보이게
②
게임을 시작할 때 0으로 맞추기
매번 처음 점수는 0부터
③
별을 받으면 1 올라가게 (다음 단계에서 연결)
지금은 "올리는 방법"만 찾아두기
💡 힌트 1 — 무엇이 필요할까?
숫자를 저장하고 화면에 보여주려면 무엇이 필요할까?
(→ 자료 카테고리의 '변수'를 찾아봐!)
💡 힌트 2 — 떠올려봐
파이썬에서 점수를 셀 때 숫자를 0부터 시작했었지? 엔트리에도 비슷하게 "0으로 정하는" 방법이 있을 거야. 어느 카테고리일까?
🤫 약속
블록 이름은 안 적어줄게. 변수를 직접 만들어보고 어떤 블록들이 생기는지 구경해봐!
✅ 과제 체크리스트
✓
"점수" 변수를 만들었다
✓
화면에 점수 숫자가 보인다
✓
시작할 때 점수가 0이 되게 했다
✓
점수를 1 올리는 블록을 찾아뒀다
✏️ 이 블록이 뭐 하는 건지 말로 설명해보기
'변수'가 뭔지, 그리고 왜 시작할 때 0으로 맞춰야 하는지 말로 설명해봐.
🆘 막혔을 때
블록을 못 찾겠으면 블록 모음 위쪽 검색창에 "변수", "정하기", "더하기" 같은 단어를 일부만 쳐봐.
⭐
단계 3
별 떨어뜨리기
별이 위에서 계속 떨어지게!
⏱ 15분
🎯 미션
별이 위에서 계속 떨어지게 하자.
📋 과제 — 이걸 해내면 성공!
①
별이 위에서 아래로 내려오기
위쪽에서 시작해서 아래로
②
별이 무작위(랜덤) 위치에서 나오기
매번 다른 자리에서 떨어지게
③
별이 계속 새로 나타나기
한 개만 떨어지고 끝나면 안 돼!
💡 힌트 1 — 예전 게임 떠올리기
예전에 만든 '피하기 게임'에서 폭탄을 어떻게 계속 만들었지? 그 방법을 떠올려봐. (복제!)
💡 힌트 2 — 한 자리만?
별이 항상 같은 자리에서만 떨어진다면? 무작위 수 블록을 찾아서 위치에 끼워봐. 어느 카테고리에 있을까? (→ 계산)
💡 힌트 3 — 깔끔하게
원본 별은 안 보이게 숨기고, 복제본만 떨어지게 하면 깔끔해. 어떻게 숨길까? (→ 생김새 카테고리)
✅ 과제 체크리스트
✓
별이 위에서 아래로 내려온다
✓
별이 매번 다른(무작위) 위치에서 나온다
✓
별이 계속 새로 떨어진다
✓
▶ 실행해서 별 비가 내리는지 확인했다
✏️ 이 블록이 뭐 하는 건지 말로 설명해보기
'복제'가 무슨 뜻인지, 그리고 '무작위 수'를 왜 썼는지 말로 설명해봐.
🆘 막혔을 때 — 3단계 규칙
① 10분 혼자 → 피하기 게임에서 폭탄 만들던 흐름 떠올리기
② AI에 힌트만 → "엔트리에서 똑같은 걸 계속 만들려면 어떤 종류의 블록이 필요해? 정답 말고 힌트만!"
③ 직접 끼우고 설명
🏆
단계 4
별 받기·완성
별을 받으면 점수 +1! 게임 완성!
⏱ 15분
🎯 미션
별이 바구니에 닿으면 점수 +1!
📋 과제 — 이걸 해내면 완성!
①
별이 바구니에 닿았는지 확인하기
닿았을 때만 점수가 올라야 해
②
닿으면 점수를 올리기
2단계에서 찾아둔 "점수 올리기"를 여기 연결!
③
받은 별은 사라지게 하기
안 그러면 점수가 마구 올라가!
💡 힌트 1 — 닿았는지 어떻게 알까?
별이 바구니에 닿았는지 어떻게 알 수 있을까?
(→ 판단 카테고리의 '닿았는가?'를 찾아봐!)
💡 힌트 2 — 누구한테 닿았지?
'닿았는가?' 블록은 처음에 다른 걸 가리킬 수도 있어. 드롭다운을 눌러서 "바구니"로 바꿔야 할까? 직접 눌러서 확인해봐!
💡 힌트 3 — 한 번만 올라야 해
왜 받은 별을 사라지게 해야 할까? 안 사라지면 점수가 어떻게 될지 상상해봐. (계속 닿아 있으면?)
🤖 AI에 물어보는 좋은 예 — 학습용
❌ 이렇게 묻지 마
"별 받기 게임 코드 전부 알려줘"
✅ 이렇게 물어
"엔트리에서 키를 누르면 캐릭터가 움직이게 하려면 어떤 블록이 필요해? 정답 말고 힌트만!"
✅ 완성 체크리스트
✓
별이 바구니에 닿았는지 확인하게 했다
✓
닿으면 점수가 1 올라간다
✓
받은 별은 사라진다
✓
▶ 실행해서 별을 받아 점수를 올려봤다!
✏️ 이 블록이 뭐 하는 건지 말로 설명해보기
'닿았는가?'를 왜 점수 올리는 데 썼는지 말로 설명해봐. (피하기 게임에서는 닿으면 나빴는데, 여기서는 왜 좋을까?)
🏆

별 받기 게임 완성!

바구니 움직이기 → 점수 → 별 떨어뜨리기 → 별 받기
네가 직접 블록을 찾아 만든 진짜 게임이야! 🌟

🆘 막혔을 때 — 3단계 규칙
① 10분 혼자 → "닿았는지 확인 → 점수 올리기 → 별 삭제" 순서를 종이에 그려보기
② AI에 힌트만 → "정답 말고, 어떤 종류의 블록이 필요한지만!"
③ 직접 만들고 설명 → 말로 설명 못 하면 다시!
