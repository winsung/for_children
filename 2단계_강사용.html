5회차 · 개념 1
print() — 화면에 출력하기
⏱ 5분
기본 형태

강사 설명 멘트

🔍 짚어줄 포인트 3가지
① 소문자 print — Print는 안 됨 (NameError)
② 괄호 ( ) — 함수의 표시. 안에 보여줄 내용을 넣음
③ 따옴표 " " — 문자열의 표시. 빠지면 SyntaxError
📝 직접 해보기 — 5문장
print("내 이름은 ___ 이야") print("___ 학교에 다녀") print("좋아하는 음식은 ___") print("좋아하는 색은 ___") print("오늘 기분은 ___")
5회차 · 개념 2
변수 — 이름표 붙은 상자
⏱ 10분
변수 만들기
name = "철수" # "name" 상자에 "철수" 담기 age = 11 # "age" 상자에 11 담기 print(name) # 상자 꺼내서 보여주기 print(age)
⚠️ 문자열 + 숫자 합치기
name = "철수" age = 11 print("안녕! 나는 " + name + " 이고 " + str(age) + "살이야")
숫자(age)를 문자열과 합칠 때는 str()로 감싸야 함.
안 하면 TypeError 발생 → 일부러 체험하게 하기 좋은 기회.
상자 비유 — 칠판에 그려주기
┌─────────┐ ┌─────────┐ │ name │ │ age │ │ "철수" │ │ 11 │ └─────────┘ └─────────┘
"이름표를 붙이면 나중에 꺼내 쓸 수 있어. 그래서 변(數)수 — 변할 수 있는 숫자."
💡 핵심 질문
"변수가 없으면 어떻게 될까?
name을 지우고 print(name) 실행해봐"
→ NameError 체험으로 자연스럽게 연결

6회차 · 개념
input() — 사람 → 컴퓨터 방향
⏱ 10분
기본 형태
name = input("이름이 뭐야? ") print("반가워, " + name + "!")
실행하면 커서가 깜빡이며 입력을 기다림. 사용자가 타이핑 후 Enter → name에 저장.
숫자 입력 받기 — int() 변환
age = input("나이를 입력해: ") # 항상 문자열로 받음 age_num = int(age) # 숫자로 변환 future_age = age_num + 10 print("10년 후엔 " + str(future_age) + "살이야!")
한 줄로 줄이는 패턴도 가르치기:
age = int(input("나이를 입력해: "))
방향 비교 표
함수
방향 / 역할
print()
컴퓨터 → 사람
화면에 보여주기
input()
사람 → 컴퓨터
키보드 입력 받기
⚠️ 함정 — input은 항상 문자열
"11" + "10" = "1110" (이어붙이기)
11 + 10 = 21 (덧셈)
→ 그래서 int() 변환이 필요함

if / elif / else + 들여쓰기 규칙
⏱ 10분
if — "만약에"
"만약 비가 오면 → 우산. 아니면 → 그냥 나가기"
weather = "비" if weather == "비": print("우산 챙겨!") else: print("그냥 나가!")
elif — 여러 조건
score = 85 if score >= 90: print("A 등급") elif score >= 80: print("B 등급") elif score >= 70: print("C 등급") else: print("더 열심히!")
⚠️ 들여쓰기 규칙
if score >= 90: print("A 등급") # 스페이스 4칸 print("끝") # 들여쓰기 없음 = if 밖
들여쓰기를 빼면 → IndentationError 발생
이 오류 보면 먼저 들여쓰기 확인하는 습관!

diff = abs(guess - secret) # 절댓값 if guess == secret: print("정답!") elif diff <= 2: print("거의 다 왔어!") else: print("많이 틀렸어!")
⚠️ 힌트만 주기 — abs() 한 줄만 알려주고 나머지는 직접 짜게.

for vs while — 언제 무엇을 쓰나

for문 — 정해진 횟수 반복
for i in range(5): # 0,1,2,3,4 (5번) print(i) for i in range(1, 6): # 1,2,3,4,5 print(i) for i in range(0, 10, 2): # 0,2,4,6,8 (2씩) print(i)

상황

while
언제 끝날지 모를 때
무한 반복 + 탈출
while True + break
게임 루프

해결: 빨간 줄 위치부터 확인. 오타·따옴표·괄호·콜론 순으로 체크.

해결: 변수 이름 스펠링 확인. 정의 위치도 확인.
9회차 · 개념 2
오류 4총사 (2) — Indentation / Type
⏱ 7분
③ IndentationError — 들여쓰기

해결: 들여쓰기 칸 수가 일정한지 확인. Tab과 Space 절대 혼용 금지.

해결: 다른 타입 연산 시 str() 또는 int() 변환.

import random def show_status(hp, items): print("\n=== 현재 상태 ===") print("HP: " + str(hp)) print("아이템: " + str(items)) def battle(enemy_name): enemy_hp = random.randint(10, 30) print(enemy_name + " 등장! (HP: " + str(enemy_hp) + ")") return enemy_hp # 게임 시작 player_name = input("영웅의 이름을 입력해: ") player_hp = 100 inventory = [] rooms = ["숲", "동굴", "마을"] print("\n" + player_name + " 의 모험이 시작됐어!") for room in rooms: print("\n=== " + room + " 도착! ===") show_status(player_hp, inventory) choice = input("1. 탐험 2. 쉬기: ") if choice == "1": enemy_hp = battle("고블린") damage = random.randint(5, 20) player_hp -= damage print(str(damage) + " 데미지!") if player_hp <= 0: print("게임 오버!") break inventory.append("포션") elif choice == "2": player_hp += 20 print("20 HP 회복!") print("\n모험 끝! 최종 HP: " + str(player_hp))
