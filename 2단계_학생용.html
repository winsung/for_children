정답을 7로 정해놓고, 사용자가 더 크게/작게/정답인지 알려주는 게임!
📝 개념 — if 구조
⚠️ 직접 타이핑 weather = "비" if weather == "비": print("우산 챙겨!") else: print("그냥 나가!")
⚠️ 가장 헷갈리는 거
= 는 담기 (name = "철수")
== 는 비교 (name == "철수")
📝 elif — 여러 조건
⚠️ 직접 타이핑 score = 85 if score >= 90: print("A 등급") elif score >= 80: print("B 등급") elif score >= 70: print("C 등급") else: print("더 열심히!")
⚠️ 들여쓰기 — Python의 핵심
if score >= 90: print("A 등급") # 4칸 (Tab 1번) print("끝") # 들여쓰기 없음 = if 밖
들여쓰기를 빼면 → IndentationError!
이 오류 보면 먼저 들여쓰기를 확인해.

⚠️ 오류 4총사 — 외워두기
① SyntaxError — 문법
print("Hello) # 따옴표 빠짐 if age > 10 # : 빠짐
해결: 빨간 줄 위치부터 따옴표·괄호·콜론 확인

nane = "철수" print(name) # nane이랑 name이 다름
해결: 변수 이름 스펠링 확인
③ IndentationError — 들여쓰기
if age > 10: print("ok") # 들여쓰기 없음
해결: Tab과 Space 섞지 않기. 칸 수 통일
④ TypeError — 타입
print("나이: " + age) # 글자 + 숫자 ❌ print("나이: " + str(age)) # 해결 ✅
해결: str() 또는 int()로 타입 변환

함수 = 레시피
라면 끓이기 방법을 한 번 적어두면, 매번 다시 적지 않고 "라면 끓이기" 한마디로 끝!
📝 함수의 구조
def 함수이름(매개변수1, 매개변수2): # 함수 내용 (들여쓰기 필수) return 결과값 # 호출 결과 = 함수이름(값1, 값2)
📝 예시 — 인사 함수
⚠️ 직접 타이핑 def introduce(name, age): print("=== " + name + " ===") print("나이: " + str(age)) introduce("철수", 11) introduce("영희", 12)
두 명, 세 명, 백 명도 한 줄씩으로 인사 가능!
📝 return — 값 돌려주기
# return 없음 — 출력만 def greet(name): print("안녕, " + name) # return 있음 — 값을 받아서 다시 쓸 수 있음 def add(a, b): return a + b result = add(3, 5) # result = 8 print(result)
🚀 실습 — 계산기 함수 (30분)
⚠️ 손으로 타이핑 def add(a, b): return a + b def subtract(a, b): return a - b def multiply(a, b): return a * b def divide(a, b): if b == 0: return "0으로 나눌 수 없어!" return a / b print(add(10, 5)) print(divide(10, 0))

서랍이 여러 개 있고, 각 서랍에 번호가 붙어있어.
0번 서랍, 1번 서랍, 2번 서랍 ... (0부터 시작!)
📝 변수 vs 리스트
⚠️ 직접 타이핑 # 변수 3개는 불편 name1 = "철수" name2 = "영희" name3 = "민수" # 리스트로 묶기 — 한 줄! friends = ["철수", "영희", "민수"]
📝 기본 조작
fruits = ["사과", "바나나", "딸기"] print(fruits[0]) # 사과 (0번) print(fruits[-1]) # 딸기 (마지막) print(len(fruits)) # 3 (길이) fruits.append("포도") # 끝에 추가 fruits.remove("바나나") # 삭제 print("사과" in fruits) # True/False
📝 리스트 + for문
friends = ["철수", "영희", "민수"] for friend in friends: print("안녕, " + friend) # 번호와 함께 for i, friend in enumerate(friends): print(str(i+1) + ". " + friend)
